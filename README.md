# API Call Retry
This app is a way to demonstrate how to retry API call automatically and revert previous calls when the current fails.

## Built With

- Node
- Express
- Jest
- Chai
- JavaScript

## Getting Started


To get a local copy up and running follow these simple example steps in the command line:

1. Clone the repo: `https://github.com/HENRYKC24/Neurum-Health-Node-Test-App.git`

2. Go to the project folder: `cd Neurum-Health-Node-Test-App`

3. Install dependencies: `npm install`

4. Start application locally: `npm start`
